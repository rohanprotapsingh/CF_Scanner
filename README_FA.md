## ✨ ویژگی‌ها

| ویژگی | توضیح |
|-------|-------|
| 🌐 **رابط وب** | رابط کاربری زیبا با تم تاریک و ریسپانسیو |
| 🔍 **دو حالت اسکن** | اسکن آی‌پی تمیز (بدون کانفیگ) + اسکن با کانفیگ |
| 📡 **VLESS + Trojan** | پشتیبانی کامل از هر دو پروتکل |
| 📱 **بیش از ۲۰ اپراتور** | رنج آی‌پی از پیش تنظیم شده برای اپراتورهای ایران |
| ✏️ **رنج سفارشی** | امکان وارد کردن رنج CIDR دلخواه |
| ⚡ **اسکن غیرهمزمان** | اسکن همزمان تا ۲۰۰ آی‌پی |
| 📊 **پیشرفت زنده** | نوار پیشرفت، لاگ و آمار لحظه‌ای |
| 🏆 **امتیازدهی هوشمند** | امتیاز بر اساس پینگ + سرعت |
| 📋 **کپی با یک کلیک** | کپی تکی آی‌پی یا همه نتایج |
| 📥 **خروجی** | دانلود به فرمت کانفیگ، لیست IP یا JSON |
| 🤖 **ربات تلگرام** | ارسال مستقیم نتایج به تلگرام |
| 💾 **ذخیره تنظیمات** | ذخیره امن اطلاعات ربات تلگرام |
| 🔄 **میرور خودکار** | پیدا کردن خودکار API تلگرام فعال |

## 📦 نصب

### پیش‌نیازها
- Python 3.8+
- pip

```bash

git clone https://github.com/rohanprotapsingh/CF-Scanner.git
cd CF-Scanner

# نصب وابستگی‌ها
pip install -r requirements.txt

# اجرا
python app.py
