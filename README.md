## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌐 **Web Interface** | Beautiful dark-themed responsive web UI |
| 🔍 **Two Scan Modes** | Clean IP scan (no config needed) + Config-based scan |
| 📡 **VLESS + Trojan** | Full support for both protocols |
| 📱 **20+ ISPs** | Pre-configured IP ranges for Iranian operators |
| ✏️ **Custom Ranges** | Enter your own CIDR ranges |
| ⚡ **Async Scanning** | Scan up to 200 IPs simultaneously |
| 📊 **Real-time Progress** | Live progress bar, logs, and statistics |
| 🏆 **Smart Scoring** | Score based on ping + speed |
| 📋 **One-click Copy** | Copy individual IPs or all results |
| 📥 **Export** | Download as VLESS/Trojan configs, IP list, or JSON |
| 🤖 **Telegram Bot** | Send results directly to Telegram |
| 💾 **Save Settings** | Securely save Telegram bot credentials |
| 🔄 **Auto API Mirror** | Automatically finds working Telegram API |

```bash
